# 🎓 学术生存指南 —— 通关指南

## 游戏目标

**一句话：在精力、存款、导师满意度三者全都没归零的前提下，把论文进度堆到 100%，完成答辩，光荣毕业。**

---

## 核心机制

### 四项核心资源


| 资源              | 初始值    | 归零后果          | 如何补充        |
| --------------- | ------ | ------------- | ----------- |
| ⚡ **精力**        | 80/100 | 💀 倒在实验室，游戏结束 | 摸鱼、睡好觉事件、休息 |
| 📝 **论文进度**     | 0/100  | 无法毕业          | 做实验、写论文     |
| 👨‍🏫 **导师满意度** | 60/100 | 📋 被退学，游戏结束   | 开会、出成果      |
| 💰 **存款**       | 5000   | 🏚️ 经济崩溃，游戏结束 | 做副业         |


### 三项次要资源

- 😊 **心理健康** (70/100)：影响正面/负面事件的触发概率
- 🤝 **社交资本** (30/100)：影响会议、副业等社交相关效果
- 🍀 **运气** (50/100)：影响随机事件的概率和效果

## 通关策略



### 🥇 推荐策略：稳扎稳打

1. **每月做一次实验 + 一次写论文**（每月可各执行多次）
2. 精力低于 40 时**摸鱼回精力**
3. 导师满意度低于 50 时**开一次会**
4. 存款低于 2000 时**做一次副业**
5. 以此循环，论文进度稳步增长



### 🥈 激进策略：冲刺流

1. 疯狂做实验 + 写论文，一月多次
2. 精力和存款濒临崩溃边缘
3. 赌随机事件的正面效果
4. **高风险高回报**，可能更快毕业，也可能突然暴毙



### 🎯 关键数值节点

- **精力 < 20**：危险区，立即摸鱼
- **导师满意度 < 30**：会触发更多负面事件，立即开会
- **存款 < 1000**：生存危机，做副业
- **论文进度 50**：触发从科研期过渡到写作期的条件
- **论文进度 90**：触发从写作期过渡到答辩期的条件
- **论文进度 100**：可以毕业！

1. **每3个月自动存档一次**，也可以手动存档到3个槽位
2. **每月生活费3000元，学校补助2000元**，净支出1000元

5. **运气值**会放大所有随机事件的影响，多触发正面事件可以提升运气
6. **不同学科**有不同的研究描述（但资源数值相同）
